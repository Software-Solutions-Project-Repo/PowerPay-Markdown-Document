# Security

> Note: The Security Tab in PowerPay is only accessible to users with administrative privileges. Regular users will not be able to access this tab unless granted permission by an admin.

## How to add a New Group

To add a new group, click on the Security Tab from the side menu. Click the Add Group Button on the top-left side of the page to open the **Group Management** window.

![Add Group Button](https://cguzaxwlyyyjudqllchm.supabase.co/storage/v1/object/public/Images/Security/security-add-group.png)

You are required to fill in the following details:

- Group Name
- Description

Click the Add Group Button to add the group.

## Searching for a Group

On the Group Management window, there is a search bar with a prompt to **Search Groups**.
![Search Groups Search Bar](https://cguzaxwlyyyjudqllchm.supabase.co/storage/v1/object/public/Images/Security/security-search-groups.png)

To search for a group, enter the name or description of the group in the search bar. A list of relevant groups that match the search will appear below.

![Group Search Results](https://cguzaxwlyyyjudqllchm.supabase.co/storage/v1/object/public/Images/Security/security-search-results.png)

## Editing Group Name and Description

To edit a new group, open the Security Tab from the side menu. On the security tab a list of all groups will be displayed.

![List of All Groups](https://cguzaxwlyyyjudqllchm.supabase.co/storage/v1/object/public/Images/Security/security-group-list.png)

To edit a group’s name or description, click the green button (or blue button on dark mode) next to the group, under the edit column.
![Edit Group Button](https://cguzaxwlyyyjudqllchm.supabase.co/storage/v1/object/public/Images/Security/security-edit-group.png)

Enter the desired changes, then click the Update Group Button (Denoted by a Checkmark Icon). To cancel an edit, click the Back Button.

![Update Group Button](https://cguzaxwlyyyjudqllchm.supabase.co/storage/v1/object/public/Images/Security/security-update-button.png) ![Back Button](https://cguzaxwlyyyjudqllchm.supabase.co/storage/v1/object/public/Images/Security/security-back-button.png)

### Alternative Way to Edit Groups

At the bottom of the group management window, a list of all groups will be displayed. To edit a group’s name or description, click the green button (or blue button on dark mode) next to the group. Enter the desired changes, then click the Update Group button. To cancel an edit, click the Cancel Edit button.

## Deleting a Group

To delete a group, open the group managaement window. Under the list of all groups, search for the group that you wish to delete.  
Under the Actions column next to the group, click the red button with the trash can icon to delete the group.

![Delete Button](https://cguzaxwlyyyjudqllchm.supabase.co/storage/v1/object/public/Images/Security/security-delete-group.png)

Confirm that you wish to delete the group by clicking **DELETE**

## Adding a new System Module

To add a new System Module, click on the Security Tab from the side menu and click the **System Modules** button to open the System Modules Management window.

![System Modules Button](https://cguzaxwlyyyjudqllchm.supabase.co/storage/v1/object/public/Images/Security/security-system-modules-button.png)

You must enter the following details:

- Application Location
- Key Code

Once finished, click the **Add System Module** button to add the module.

![Add System Modules Button](https://cguzaxwlyyyjudqllchm.supabase.co/storage/v1/object/public/Images/Security/security-add-module.png)

## Editing System Module

To edit a module’s location or key code, open the Security tab, click System Modules, and scroll down to the list of all System Modules.

![All System Modules List](https://cguzaxwlyyyjudqllchm.supabase.co/storage/v1/object/public/Images/Security/security-module-list.png)

Search for the module that you wish to edit.
Under the actions column, click the green button (or blue button on dark mode) next to the module to edit it.

![Edit Button](https://cguzaxwlyyyjudqllchm.supabase.co/storage/v1/object/public/Images/Security/security-edit-button.png)

Enter the desired changes, then click Update System Module.

To cancel an edit, click Cancel Edit.

## Deleting a System Module

To delete a system module, open the Security tab, click System Modules and scroll down to the list of all System Modules and search for the module that you wish to delete.

Under the Actions Column, Click the red button with the trash can icon next to a module to delete it.

![Delete Button](https://cguzaxwlyyyjudqllchm.supabase.co/storage/v1/object/public/Images/Security/security-delete-group.png)

Confirm that you wish to delete the module by clicking **DELETE**
