# Reports

## How to Generate a Report

To generate a new Report, open the Reports tab in the side menu. A list of all report templates will be displayed with the Report’s Category, Description, File Name, Version Number and Parameters. Under the column labelled Actions, click the information icon for the report that you wish to generate.  
![][image7]  

 A window will open that displays the details of the report. Select the desired Report Format (PDF, Excel, Word Document) and Network (Internet, Intranet). Fill out all other details for the report, then click Generate Report at the bottom of the window to generate the report.  
![][image8]  

Note: Other details depend on the type of report you are generating. (e.g. Start Date and End Date for employeelist.rpt)
