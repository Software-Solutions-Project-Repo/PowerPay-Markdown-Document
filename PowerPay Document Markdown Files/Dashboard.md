# Dashboard

In PowerPay, the Dashboard tab provides an overview of each process in the PowerPay system. To access the Dashboard, open the side menu and select Dashboard.

## Overview

At the top of the Dashboard, there are four clickable tabs, which include: Active Employees, Next Payroll, Pending Approvals, and Current Period.

**Active Employees Button** link that redirects the user to the Employee Search tab when clicked.
![Active Employees Button](https://cguzaxwlyyyjudqllchm.supabase.co/storage/v1/object/public/Images/Dashboard/dashboard-active-employees-button.png)


**Next Payroll Button** provides a quick link that redirects the user to the Payroll Sheets tab when clicked.
![Next Payroll Button](https://cguzaxwlyyyjudqllchm.supabase.co/storage/v1/object/public/Images/Dashboard/dashboard-next-payroll-button.png)

**Pending Approvals Button** provides a quick link that displays all of the user’s unread notifications when clicked.
![Pending Approvals Button](https://cguzaxwlyyyjudqllchm.supabase.co/storage/v1/object/public/Images/Dashboard/dashboard-pending-approvals-button.png)

**Current Period Button** displays the current payroll period. It also contains a quick link that redirects the user to the Payroll Cycles tab when clicked.
![Current Period Button](https://cguzaxwlyyyjudqllchm.supabase.co/storage/v1/object/public/Images/Dashboard/dashboard-current-period-button.png)

**Notifications:** To view your notifications, click the bell icon at the top right of your screen.

![Notification Bell Icon](https://cguzaxwlyyyjudqllchm.supabase.co/storage/v1/object/public/Images/Dashboard/dashboard-notification-icon.png)

**Profile:** To view your profile, click on your profile picture at the top right of your screen.

![Profile Icon Button](https://cguzaxwlyyyjudqllchm.supabase.co/storage/v1/object/public/Images/Dashboard/dashboard-profile-picture.png)

## Recent Activity  
Recent Activity displays a list of notifications for system changes or actions that the user may need to complete. Each item listed contains a short description (e.g. “March 2024 payroll completed for 254 employees.”) and a timestamp (e.g. 2 hours ago, a day ago)
![Recent Activity List](https://cguzaxwlyyyjudqllchm.supabase.co/storage/v1/object/public/Images/Dashboard/dashboard-recent-activity.png)

To view a list of all Recent Activity notifications, click the View All button on the top right of the Recent Activity section on the dashboard tab.   
![View All Button](https://cguzaxwlyyyjudqllchm.supabase.co/storage/v1/object/public/Images/Dashboard/dashboard-view-all-button.png)
