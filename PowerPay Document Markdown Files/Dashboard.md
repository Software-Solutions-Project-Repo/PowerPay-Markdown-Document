# Dashboard

In PowerPay, the Dashboard tab provides an overview of each process in the PowerPay system. To access the Dashboard, open the side menu and select Dashboard.

## Overview

At the top of the Dashboard, there are four clickable tabs, which include: Active Employees, Next Payroll, Pending Approvals, and Current Period.

**Active Employees Button** link that redirects the user to the Employee Search tab when clicked.

**Next Payroll Button** provides a quick link that redirects the user to the Payroll Sheets tab when clicked.

**Pending Approvals Button** provides a quick link that displays all of the user’s unread notifications when clicked.

**Current Period Button** displays the current payroll period. It also contains a quick link that redirects the user to the Payroll Cycles tab when clicked.

**Notifications:** To view your notifications, click the bell icon at the top right of your screen.

**Profile:** To view your profile, click on your profile picture at the top right of your screen.

## Recent Activity  
Recent Activity displays a list of notifications for system changes or actions that the user may need to complete. Each item listed contains a short description (e.g. “March 2024 payroll completed for 254 employees.”) and a timestamp (e.g. 2 hours ago, a day ago)
