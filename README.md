# PowerPay-Markdown-Document

This repository contains manuals for the PowerPay application stored as markdown files.

Each file represents the following chapters of manual:

- Dashboard: `Dashboard.md`
- Payroll Sheets: `Payroll Sheets.md`
- Payroll Cycles: `Payroll Cycles.md`
- Employee Search: `Employee Search.md`
- Template Manager: `Template Manager.md`
- Reports: `Reports.md`
- Configuration: `Configuration.md`
- Security: `Security.md`

These files should be uploaded to the database through the RAG Document Manager so that they can be stored as vector embeddings and used by the Payroll RAG.
